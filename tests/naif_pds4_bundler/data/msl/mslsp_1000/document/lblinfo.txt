MSL SPICE Data PDS Labels
=========================

   PDS labels for MSL SPICE kernel files contained in this data set are
   stored as detached files in the same directories where the actual
   data files are. The naming convention for the label files in this
   data set is the following: for a particular data file the label file
   has exactly the same name with extension ``.lbl''.
