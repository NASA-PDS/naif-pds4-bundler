SPICE System Toolkit
====================

   The latest SPICE Toolkit for a variety of computer platforms is
   available at the NAIF Node of the PDS electronically via anonymous
   FTP:

      ftp://naif.jpl.nasa.gov/pub/naif/toolkit

   A few additional utility programs for making and modifying SPICE
   kernels, including a tool (BINGO) that can converts either binary or
   text kernels from one format to another, are available at the NAIF
   Node of the PDS electronically via anonymous FTP:

      ftp://naif.jpl.nasa.gov/pub/naif/utilities

   The latest complete set of SPICE System tutorials is available at
   the NAIF Node of PDS electronically via anonymous FTP:

      ftp://naif.jpl.nasa.gov/pub/naif/toolkit_docs/Tutorials

   The latest complete set of ``hands-on'' lessons to assist new users
   in learning fundamentals of programming with SPICE modules is
   available at the NAIF Node of PDS electronically via anonymous FTP:

      ftp://naif.jpl.nasa.gov/pub/naif/toolkit_docs/Lessons
