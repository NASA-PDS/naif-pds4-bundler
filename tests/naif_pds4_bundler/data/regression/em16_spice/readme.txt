Bundle Description for the ExoMars 2016 SPICE Archive Bundle


Overview
========

   The ExoMars 2016 SPICE archive bundle contains observation geometry and
   other ancillary data in the form of SPICE System kernel files for
   the ExoMars 2016 spacecraft, its instruments, and targets.

   The detailed description of the ExoMars 2016 SPICE archive is provided
   in the latest version of spiceds_v*.html included in the
   document collection.


Document Collection
===================

   The document collection contains documents describing the ExoMars 2016
   SPICE archive, PDS4 collection labels, and PDS4 collection
   inventory tables.


SPICE Kernels Collection
========================

   The SPICE Kernels collection contains ExoMars 2016 SPICE kernels and
   meta-kernels, PDS4 collection labels, and PDS4 collection inventory tables.
