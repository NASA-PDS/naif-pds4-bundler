Bundle Description for the Venus Climate Orbiter Akatsuki SPICE Archive Bundle
==============================================================================

Overview
========


   The VCO SPICE archive bundle contains observation geometry
   and other ancillary data in the form of SPICE System kernel files
   for the Venus Climate Orbiter Akatsuki (VCO, also known as PLANET-C
   and AKATSUKI) spacecraft, its instruments, and targets.

   The detailed description of the VCO SPICE archive is provided in the
   latest version of spiceds_v*.html included in the document collection.


Document Collection
===================

   The document collection contains documents describing the
   VCO SPICE archive, PDS4 collection labels, and PDS4
   collection inventory tables.


SPICE Kernels Collection
========================

   The SPICE Kernels collection contains VCO SPICE kernels and
   meta-kernels, PDS4 collection labels, and PDS4 collection
   inventory tables.


Miscellaneous Collection
========================

   The miscellaneous collection contains the MD5 checksum list of
   all files in the VCO SPICE archive bundle and the VCO orbit
   number files.


References
==========

   For more information about the SPICE system see documentation and
   tutorials available at the NAIF Node of the PDS:

      http://naif.jpl.nasa.gov
