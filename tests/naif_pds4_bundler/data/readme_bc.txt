Bundle Description for the BepiColombo SPICE Archive Bundle


Overview
========

   The BepiColombo SPICE archive bundle contains observation geometry and
   other ancillary data in the form of SPICE System kernel files for
   the BepiColombo spacecraft, its instruments, and targets.

   The detailed description of the BepiColombo SPICE archive is provided
   in the latest version of spiceds_v*.html included in the
   document collection.


Document Collection
===================

   The document collection contains documents describing the BepiColombo
   SPICE archive, PDS4 collection labels, and PDS4 collection
   inventory tables.


SPICE Kernels Collection
========================

   The SPICE Kernels collection contains BepiColombo SPICE kernels and
   meta-kernels, PDS4 collection labels, and PDS4 collection inventory tables.


References
